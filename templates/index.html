Flask-Portfolio

This personal portfolio website is a dynamic and responsive platform that showcases my skills, projects, and experiences. It features a clean and user-friendly design with both light and dark themes for optimal user experience. Below, you can find an overview of the technologies used:
Technologies Used:
Frontend:
HTML: foundation of the website's structure and content. CSS: Styling the website to create an attractive and visually appealing layout. JS (JavaScript): Enhancing the user experience with interactivity and dynamic content.
Backend:
Python: The primary programming language used for server-side logic and dynamic content generation. sqlite3: A lightweight, serverless database used to store and manage data related to the project, experience, and more.

Requirements: In order to run and deploy this Flask-based portfolio website, you'll need the following Python packages and tools:
flask: The Flask framework for web development. requests: A library for making HTTP requests to fetch data or interact with APIs. flask-migrate: A Flask extension for database migrations, useful with SQLAlchemy or similar database systems. waitress: A production-quality pure-Python WSGI server. gunicorn: A popular WSGI HTTP server for deploying Python web applications..
